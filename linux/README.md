# CommandLineFPS - Linux Version

![Screenshot](screenshot.png)
## Prerequisites

* CMake - C/C++ build system
* NCursesw - Library for interacting with the terminal

### Installation

* Ubuntu: `sudo apt-get install cmake libncursesw5-dev`
* Other distros: `¯\_(ツ)_/¯`

## Build

```sh
mkdir build
cd build
cmake ..
make
```

## Run

`./CommandLineFPS`
